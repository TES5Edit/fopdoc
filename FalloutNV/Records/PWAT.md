PWAT
====

Placeable Water

## Format

Count | Subrecord | Name | Type | Info
------|-------|------|------|-----
+ | EDID | Editor ID | cstring |
+ | [OBND](Subrecords/OBND.html) | Object Bounds | struct |
+ | | [Model Data](Subrecords/Model.html) | collection |
+ | DNAM | | struct
 
### DNAM

Name | Type | Info

Flags | uint32 | See below for values.
