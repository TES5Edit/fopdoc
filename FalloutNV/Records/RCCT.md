RCCT
====

Recipe Category

## Format

Count | Subrecord | Name | Type | Info
------|-----------|------|------|-----
+ | EDID | Editor ID | cstring |
 | FULL | Name | cstring |
 | DATA | Flags | uint8 | See values below.

### DATA Values

Value | Meaning
------|--------
0x01 | Subcategory
0x02 | ??
0x04 | ??
0x08 | ??
0x10 | ??
0x20 | ??
0x40 | ??
0x80 | ??
