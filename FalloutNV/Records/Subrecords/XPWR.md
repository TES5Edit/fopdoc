XPWR Subrecord
==========

## Format

Name | Type | Info
-----|------|-----
Reference | formid | FormID of a [REFR](../REFR.md) record.
Type | uint32 | Flags - see below for values.
 
### Type Flag Values

Value | Meaning
------|--------
0x00000001 | Reflection
0x00000002 | Refraction
