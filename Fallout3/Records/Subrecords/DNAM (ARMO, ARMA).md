DNAM Subrecord
==========

As used in the [ARMO](../ARMO.html) and [ARMA](../ARMA.html) record types.

## Format

Name | Type | Info
-----|------|-----
AR | int16 | Value is divided by 100 by game.
Flags | uint16 | See below for details.
 
#### Flag Values

Value | Meaning
-----|--------
0x0001 | Modulates Voice
