GLOB
====

Global Variable

## Format

Count | Subrecord | Name | Type | Info
------|-------|------|------|-----
+ | EDID | Editor ID | cstring |
+ | FNAM | Type | uint8 | Values of `s`, `l` and `f` denote `short`, `long` and `float` variable types respectively.
+ | FLTV | Value | float32 |
