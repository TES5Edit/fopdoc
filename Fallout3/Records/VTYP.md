VTYP
====

Voice Type

## Format

Count | Subrecord | Name | Type | Info
------|-------|------|------|-----
+ | EDID | Editor ID | cstring |
+ | DNAM | Flags | uint8 | See below for values.

### DNAM Flag Values

Value | Meaning
------|--------
0x01 | Allow Default Dialog
0x02 | Female
