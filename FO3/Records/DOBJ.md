DOBJ
====

Default Object Manager

## Format

Count | Field | Name | Type | Info
------|-------|------|------|-----
+ | EDID | Editor ID | cstring |
+* | DATA | Default Object | formid | Each DATA field is mapped to a different type of object. The mapping is given below.

### DATA Mapping
